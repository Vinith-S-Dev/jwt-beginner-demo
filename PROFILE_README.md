## 🏆 Featured Project

### 🛡️ [NextGen Secure Auth Portal](https://github.com/Vinith-S-Dev/Project-folder) (FastAPI + JWT + Glassmorphism UI)
An ultra-modern, production-ready JWT Authentication Portal that implements robust user validation and Role-Based Access Control (RBAC).
* **Robust Backend:** FastAPI, Passlib (Bcrypt password hashing), Python-Jose (JWT generation and verification).
* **Interactive Frontend:** Vanilla CSS Glassmorphic design with a custom base64 token payload decoder, session countdown, and active HTTP request logger.
* **Easy Execution:** Pre-configured with Docker and `docker-compose` for local testing.
